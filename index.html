MY PROJECTS

Virtual Mouse Using Hand Gesture

Virtual mouse using hand gestures is an exciting and innovative technology that allows users to interact with computers and other devices without the need for a physical mouse or touchpad. It utilizes computer vision and gesture recognition techniques to track and interpret hand movements, enabling users to control the cursor and perform various actions through natural hand gestures.with the potential to revolutionize human-computer interaction and offer new possibilities in various industries, including gaming, healthcare and virtual reality.

Message Sharing System using Cryptography and Steganography.

The main objective of this project is to provide a secure way to share or exchange information in this digital world. This work is a strategic combination of both cryptography and steganography techniques. The combination of both techniques provides a higher level of security by using a QR code in one layer. A three-layer architecture is proposed to accomplish this task where in each layer complexity increases in terms of security.
